
# Smart Contact Management System

A simple C++ console application to manage contacts efficiently using file handling.

## 📌 Project Overview

The **Smart Contact Management System** is a lightweight, text-based contact organizer that allows users to:

- Add new contacts
- View all saved contacts
- Search for contacts by name or phone number
- Edit existing contact details
- Delete contacts
- Save all contact data in a file (`contacts.txt`) for persistence

## 🛠️ Features

- ✅ Add Contact (Name, Phone, Email, Address)
- ✅ View All Contacts
- ✅ Search Contact by Name/Phone
- ✅ Edit Contact
- ✅ Delete Contact
- ✅ Persistent Storage using File Handling

## 🧠 Technologies Used

- Language: **C++**
- Core Concepts: **Structures**, **Vectors**, **File I/O**, **Loops**, **Conditionals**

## 📂 Files Included

- `main.cpp` – The full C++ source code for the application.
- `contacts.txt` – Automatically created file to store contact information.
- `Project_Summary.pdf` or `Project_Summary.docx` – One-page documentation with project details.
